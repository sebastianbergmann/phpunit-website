Plugins for PHPUnit

Clever Test Runner

phpunit-clever-and-smart reorders your tests so that tests that failed in a previous run are executed first.

SpeedTrap

phpunit-speedtrap reports slow-running tests right in your console.

ParaTest

paratest can execute tests in parallel.

XSLT for PHPUnit

XLST template fileset which provides an easy way to view the PHPUnit XML (JUnit) logfiles in a human readable manner using a web browser (screenshot).
