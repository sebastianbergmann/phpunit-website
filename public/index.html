Welcome to PHPUnit!

PHPUnit is a programmer-oriented testing framework for PHP.
It is an instance of the xUnit architecture for unit testing frameworks.

Take the first steps Read the documentation Learn from presentations Register for training Contribute!

Current Stable Release

PHPUnit 5.1 is the current stable release series.
It became stable on December 4, 2015.
Its support ends on February 5, 2016.

You can find out what's new in PHPUnit 5.1 in the release announcement.

PHPUnit 5.1 is supported on PHP 5.6 and PHP 7.

Old Stable Release

PHPUnit 4.8 is the current old stable release series.
It became stable on August 7, 2015.
Its support ends on August 5, 2016.

You can find out what's new in PHPUnit 4.8 in the release announcement.

PHPUnit 4.8 is supported on PHP 5.3, PHP 5.4, PHP 5.5, and PHP 5.6.

The documentation has details on how to verify PHAR releases of PHPUnit using PGP signatures and SHA1 hashes.
